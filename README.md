# StockEdge — MySQL Stock Market Database 📈

A complete MySQL database project that simulates how real Indian 
stock trading platforms like Zerodha and Groww work under the hood.

## 🗄️ Database Overview
Built with 7 interconnected tables using real NSE-listed companies 
like TCS, Reliance, Infosys, HDFC Bank, and more.

## 📋 Tables
- companies — master list of NSE/BSE listed companies
- stocks — live price snapshot per company
- users — retail and institutional investors
- portfolios — current holdings with avg buy price
- transactions — every buy/sell (audit log)
- market_data — daily OHLCV candlestick data
- watchlists — stocks users are tracking

## 🧠 SQL Concepts Covered
- DDL — CREATE, ALTER, DROP, TRUNCATE
- DML — INSERT, UPDATE, DELETE
- DQL — SELECT with WHERE, ORDER BY, DISTINCT, LIKE
- Operators — Arithmetic, Comparison, Logical, Range, List
- Joins — INNER, LEFT, RIGHT, SELF, FULL (UNION)
- Subqueries
- Aggregate Functions — SUM, AVG, COUNT, MIN, MAX
- GROUP BY with HAVING
- SQL Functions — String, Math
- Views
- Stored Procedure & Stored Function
- Triggers
- Window Functions — RANK, LAG, Moving Average
- Indexes
- TCL — COMMIT, ROLLBACK
- DCL — GRANT, REVOKE
